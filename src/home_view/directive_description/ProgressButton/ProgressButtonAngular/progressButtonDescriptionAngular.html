progressButton API Documentation

import {ProgressButtonModule} from '@eluci/progressbutton-ng2';

Directive

Button with different states like initial, inprogress, done and error.

Selector: progress-button
Exported as: ProgressButtonComponent

you can easily specify color, text, icon for the button.

To start you would need to specify the state object. In state object, you can specify the state along with its color and text to be displayed for button. you can pass a json object like 

stateObject = {
    'value': 'initial',
    'text': 'upload', 
    'color': '#337ab7'
}

Usage

Basic Example

<progress-button [state]="stateObject" [disabledstate]="false" icon="publish" >
