progressButton API Documentation

<progress-button> is a button with different states like initial, inprogress, done and error.

you can easily specify color, text, icon for the button.

To start you would need to specify the state object. In state object, you can specify the state along with its color and text to be displayed for button. you can pass a json object like 

stateObject = {
    'value': 'initial',
    'text': 'upload', 
    'color': '#337ab7'
}

Usage

Basic Example

html

<progress-button state="stateObject" icon="publish" disabledstate="false" >


Example with interpolation

you can also use interpolation for setting icon and disabledstate attribute using variables like - 

icon_name = "error"
isDisabled = false

html

<progress-button state="stateObject" icon='{{icon_name}}'' disabledstate="{{isDisabled}}" >

Attributes

Parameter Type Description
state Json object

model to define state of the button with text to be displayed and its background color. It contains key/value pairs.It contains following keys -

  • value
  • text      optional
  • color    optional
disabledstate boolean

Determines whether or not to disable the progress-button
icon string

String name of icon for initial state. icons should be from angular-material-icons only
