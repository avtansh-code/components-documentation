searchSelectDropdown API Documentation

<search-select-dropdown> is a component which can contain 'n' number of child-accordion with search and select child-accordions data feature.

To start you would need to specify the accordion-data attribute.
It is an array of objects.Each object represents one child accordion.Following is a sample data 

accordion-data = [
      {
        heading: 'Accordion 1',
        selected: false,
        open: true,
        isDisabled: false,
        color: '#00f',
        data: [
        	{
              content: 'item 1',
              selected: false
            }
        ]
      },
      {
        heading: 'Accordion 2',
        selected: false,
        open: false,
        isDisabled: false,
        color: '#00f',
        data: [
        	{
              content: 'item 1',
              selected: false
            }
        ]
      },
  ]

Usage

Basic Example

<search-select-dropdown
        heading="Search..."
        no-of-item="4"
        icon-color="#00f"
        heading-color="#00f"
        accordion-data="accordionData"
        open="isOpen"
        is-disabled="isDisabled"
        open-one-at-a-time="true">
</search-select-dropdown>

Attributes

Parameter Type Description
heading string

Placeholder text for dropdown heading.

accordion-data object[]

model to define the child accordion data. It is an array of json objects containing each accordion data.

open boolean

model to control open/close action of dropdown.

open-one-at-a-time boolean

control whether expanding one child accordion will cause other child accordions to close
is-disabled boolean

determines whether to disable the dropdown or not

heading-color string

text and background color for heading of dropdown. It should be defined in hex format
icon-color string

specifies search icon color
no-of-item number

control number of rows displayed in child accordions
