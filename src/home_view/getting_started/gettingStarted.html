How To Start ?

Include Angular Js Material and Bootstrap Library in your project either through CDN links or npm. You can see the available packages list at private npm registry, here

Installing Reusable Components

You can install the components in your local project using npm

First,Go to root directory of your project and update npmrc config file. Open npmrc file: vim ~/.npmrc
Add following lines to npmrc file for setting private and default npm registry
  • registry=http://registry.npmjs.org/
  • @eluci:registry=http://54.245.179.143:81/
Then install it
  • npm install @eluci/< component-package-name > --save
  • require('@eluci/< component-package-name >') from your code and add as a dependency for your app.

    •  const component = require('@eluci/< component-package-name >');

 angular.module('myModule',[component]);
Note:- The package might also have required css files. So, in such cases you can use a css loader in your project, if not already used. you can also see browserify-css
